MYSTERY:
  This function returns the n-th fibonacci number, where n is the input of the program. The
  interface is as follows: ./mystery <integer>. The integer must be non-negative in order for
  the function to work properly.

FUNCTIONS:
  add: Takes two parameters, and returns their sum as an integer.

  dothething: Takes a parameter, n, and returns the n-th fibonacci number recursively.
              The returned value will be contained in an integer.
